Get Started in Minutes

Launch Dreamer-V in Google Cloud Shell and start creating. It's the fastest way to experience the full potential of Veo.

gcloud config set project veo-testing GCP_REGION=us-central1 BUCKET=gs://dreamer-v-data-${GOOGLE_CLOUD_PROJECT}-${USER} gsutil mb ${BUCKET} mkdir -p dreamer-v-data docker run -d --name dreamer-v-app \ -p 8080:80 \ -e GCP_PROJECT_ID=${GOOGLE_CLOUD_PROJECT} \ -e GCP_REGION=${GCP_REGION} \ -e VIDEO_GCS_BUCKET=${BUCKET} \ -v "$(pwd)/dreamer-v-data":/app/backend/data \ --rm \ nicosoft/dreamer-v:latest
